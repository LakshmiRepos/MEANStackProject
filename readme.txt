    Extract this project into a specific folder.
    Install node, npm and MongoDB in your local Machine.
    Navigate to the root of the project, then run the following command:
    > npm install(To install all node dependencies)
    Import 'products' list to mongoDB by running the following command:
    > mongoimport -d shoppingCart -c products --file products.bson
    Once you installed all the dependencies, run 'npm start' to run the node server 
    and go to browser and type url 'localhost:3000' to work with the application.
